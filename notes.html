Notes on HTML Tags

<alt>
Used to help persons who need accessibilty technology to use their computer.
<a>
The <a> tag is used to create a link from one page to another. Needs an href attribute to function properly.
<form>
Used to collect information from website users. Sends and recieves information to the website. Requires inputs to function.
<nav>
This tag is used to seperate navigational links.
<blockquote>
Used to seperate a section that contains a quote.
